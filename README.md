# Climate Data Analysis
Weather and climate data analysis and visualization using Python, and libs pandas and matplotlib.



## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Files
- `weather_analysis.ipynb`: Data cleaning, analysis, and visualization
- `weather_data.xlsx`: Weather and climate dataset

## What This Project Shows
- Data cleaning from Excel files
- Exploratory data analysis
- Weather and climate visualizations
